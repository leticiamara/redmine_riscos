REDRISK
======================
##Instalação##
No terminal digite os seguintes comando
```shell
cd redmine/plugins
git clone https://github.com/leticiamara/redmine_riscos.git
rake redmine:plugins:migrate
```

##Versões suportadas##
- 2.3.2
- 2.3.3
